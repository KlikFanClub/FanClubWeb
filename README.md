## Installation and Setup

* Git clone https://github.com/KlikFanClub/FanClubWeb.git

* Install Composer if you don't have already globally installed
```
$ composer install
```

* Or make an Update
```
$ composer update
```

* To compile laravel
```
$ npm run watch
```

* Generate key env

* Compile laravel run watch
```
$ npm run watch
```
